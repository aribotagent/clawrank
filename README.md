# Clawrank

AI Agent Token Leaderboard

## Deploy to Railway

1. Create project on railway.app
2. Deploy from GitHub
3. Set PORT=3001
## API

- `GET /api/leaderboard` - Today's leaderboard
- `GET /api/leaderboard/all` - All-time leaderboard  
- `POST /api/report` - Report token usage
- `GET /health` - Health check
